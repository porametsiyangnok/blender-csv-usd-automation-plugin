# Blender CSV-USD Automation Plugin

🚀 IHK Final Project 2025  
Fachinformatiker für Anwendungsentwicklung  

## 📌 Project Overview

This Blender plugin automates the creation of animated GLB models
from industrial simulation data exported from ISG-virtuos.

It combines:

- USD files (3D geometry)
- CSV files (movement & animation data)

into a fully animated GLB model ready for web and mobile applications.

---

## 🎯 Problem Statement

Previously, USD geometry files and CSV motion data had to be manually
combined inside Blender. This process was time-consuming, error-prone,
and difficult to automate.

This plugin fully automates that workflow.

---

## 🛠 Technologies Used

- Python
- Blender API
- USD Import
- CSV Parsing
- Quaternion to Euler conversion
- Keyframe animation
- GLB Export (glTF)

---

## ⚙ Features

- Import USD models
- Parse CSV animation data
- Automatic object mapping
- Keyframe generation
- Animation support for multi-axis robots
- Direct GLB export
- Error logging system

---

## 📦 Installation

1. Download the repository as ZIP
2. Open Blender
3. Go to Edit → Preferences → Add-ons
4. Click "Install"
5. Select the ZIP file
6. Activate the Add-on

---

## ▶ Usage

1. Select USD file
2. Select CSV file
3. Apply animation
4. Export as GLB

---

## 🏆 Result

A fully automated pipeline to generate animated,
web-ready 3D models from simulation data.
